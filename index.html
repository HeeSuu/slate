    玩家注册

    此接口用于玩家注册，所有玩家注册都需带上前缀，以免出现注册失败的情况

    请求地址

    {{url}}/player/register

    请求参数

    类型 参数 描述
    string agent 代理帐号
    string username 玩家帐号
    string nickname 玩家昵称
    string Site_code ？
    string hash 哈希值
    玩家用户名请带上前缀注册。
    用户密码需不少于6位,部分平台对用户密码进行限制，建议6-12位，不含特殊符号。

    返回参数

    类型 参数 描述
    string playe 注册玩家用户名
    Timestamp time 账号创建时间

    游戏登录链接

    此接口用于生成玩家 登录游戏链接

    请求地址

    {{url}}//player/login

    请求参数

    类型 参数 描述
    string agent 代理帐号
    string username 玩家帐号
    string Game_code 游戏代码
    string hash 哈希值

    返回参数

    类型 参数 描述
    string player 玩家用户名
    string url 游戏登录链接
    bool new_player 布尔变量
    Timestamp time 当前时间

    返回参数

    类型 参数 描述
